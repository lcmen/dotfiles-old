# Collection of my dotfiles

## Installation

1. If using MacOS: install Homebrew and run `setup.sh`
3. Run 'rake install' from root of the project.
4. If using Logitech mouse: `sudo ln -s logiops/logid.cfg/symlink /etc/logid.cfg`
